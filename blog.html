Question-1: What are the different ways to select an element in the DOM?

There are five ways to select an element from the DOM.

The ways are:

  • getElementsByTagName,
  • getElementsByClassName
  • getElementById
  • querySelector
  • querySelectorAll

Question-2: What is the difference between innerHTML, innerText, and textContent ?

1. innerHTML: innerHTML can be used to access the text and tags inside an element.

2. innerText: innerText returns only the text inside the element, not any hidden text.

3. textContent: textContent returns all the text contained within an element and all its child elements.

Question-3: What is event delegation in the DOM?

Event delegation is a method of handling child elements by setting an event listener on a parent element. Event delegation uses less memory space, resulting in much better performance.

Question-4: What is event bubbling in the DOM?

Event bubbling is the process by which an event occurs in a child element, then in its parent element, from the parent element to the grandparent element, and then up to the document.

For example - button > div > body > document

Question-5: How do you create, add, and remove elements using JavaScript?

1. Create: To create a new element in JavaScript, we need to use the "document.createElement(tagName)"

1. Add: To add a new element in JavaScript, we need to use the "parent.appendChild(newElement)"

1. Remove: To remove element in JavaScript, we need to use the "parent.removeChild(child)"
